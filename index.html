Mini Algebra Lesson

Solving One-Step Linear Equations

Use this simple page to give learners a short pre‑test, three tiny lessons, and then a post‑test on solving one‑step equations.

Lesson Flow

How to Use This Page

  1. Open the Pre‑Test and answer the 5 questions.
  2. Read the three short modules below (and watch the example videos, if included).
  3. Have learners work a few practice problems on paper or a whiteboard.
  4. Open the Post‑Test and answer the same 5 questions again.

Tip: Use the same 5 questions for both pre‑ and post‑tests so you can measure learning gain directly.
Module 1 – What Are We Trying to Do?
Goal: Understand that solving an equation means getting the variable by itself on one side of the equal sign (for example, turning x - 3 = 8 into x = 11).
Module 2 – Using Addition & Subtraction
Show one or two worked examples such as x + 5 = 12 and y - 4 = 10, writing the “undo” step and the answer.
Module 3 – Using Multiplication & Division
Show worked examples such as 3a = 15 and b / 2 = 8, again showing the inverse operation and final answer.

Overview

Learning Objectives

  • LO1Identify the goal of solving a basic equation.
  • LO2Solve one‑step equations using addition or subtraction.
  • LO3Solve one‑step equations using multiplication or division.

Pre‑ / Post‑Test Items

  • Q1Concept: main goal when solving an equation like x - 3 = 8.
  • Q2Compute: solve x + 5 = 12.
  • Q3Compute: solve y - 4 = 10.
  • Q4Compute: solve 3a = 15.
  • Q5Compute: solve b / 2 = 8.
